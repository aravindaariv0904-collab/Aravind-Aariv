The interactive developer portfolio site of **Aravind Aariv** showcasing a fully audited, refactored, and verified suite of 11 repositories representing production-grade AI applications, full-stack monorepos, and visual audio engines.

---

## 🎨 Core Features
- **Aesthetic Glassmorphism Theme**: Floating radial background glow orbs and cursor-tilt parameters that bring pages to life.
- **Dynamic Audited Repositories**: Full interactive catalog showcasing details and code linkages of all 11 projects.
- **Experience Timeline**: Visually tracks Aravind's engineering timeline from Full-Stack Architect to Founding AI Engineer.
- **Interactive Portfolio Agent**: Integrated chatbot assistant answering recruiter queries about Aravind's projects and skills in real-time.

---

## 🏗️ Tech Stack
- **Framework**: React 19, Vite
- **Iconography**: Lucide React
- **Styles**: Glassmorphic Custom CSS

---

## 🚀 How to Run Locally

### Prerequisites
Make sure you have Node.js 18+ installed on your system.

### Installation & Execution
```bash
git clone https://github.com/aravindaariv0904-collab/Aravind-Aariv.git
cd Aravind-Aariv
npm install
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your web browser.

### Run Production Build
```bash
npm run build
```
